October Meeting: Adam Rabung & The Scala Web Framework Play

Details

  • Date: Tuesday, October 8
  • Time: 18:00-20:00
  • Place: 804RVA 1657 W. Broad St. #3, Richmond, VA (map)
  • RSVP: Meetup.com

An introduction to Scala and the Play Framework

This talk will present a quick introduction to Scala syntax & concepts, followed by an overview of building web sites/services with the Play Framework, which was recently re-written in Scala. Finally, we'll talk about Play's most interesting capabilities: asynchronous and "reactive" programming.

Adam Rabung

Adam has been a Java and Web programmer since the late 90s. He has been gradually switching to Scala over the last few years, and hopes he can avoid Java from here on out.
